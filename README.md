# Real Estate Website using React JS & Boostrap

### Technologies used in the project: 
 - React js
 - React Routes
 - JSX
 - Boostrap
 - React Tabs
 - React Widgets

### Functionalities added to the website:
 - Advanced Search Filtering
 - Drag & Drop
 - Add to favourites
 - Google Maps
